Inspect package.json on GitHub

What does it do?

The extension works on github.com whenever you navigate to a package.json file. It finds the dependencies sections and displays information such as package description, version and links related to dependencies. (Dependencies referenced by "workspace" or "github" are excluded for simplicity.)

Usage

  • Navigate to a package.json on github.com (Example)
  • Click on a dependency in "dependencies", "devDependencies", etc.
  • See the "Package Info" section in the popup.
