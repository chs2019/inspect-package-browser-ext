Inspect package.json on GitHub

What does it do?

The extension works on github.com. It displays details on dependencies found in package.json files, such as package descriptions and links to the package.

How does it work?

  • Navigate to a package.json on github. (Example)
  • Click on a dependency in "dependencies", "devDependencies", etc.
  • See the "Package Info" section in the popup.
